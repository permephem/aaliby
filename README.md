# Aaliby Auto Repair and Towing

Modern website for [Aaliby Auto Repair and Towing, LLC](https://www.aalibyautorepairandtowing.com/) — built with Next.js, React, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Production build
- `npm run start` — Start production server
- `npm run lint` — Run ESLint

## Customization

Business details (phone, email, hours, services) are centralized in `src/lib/constants.ts`. Replace gallery placeholder gradients with real photos in `src/components/Gallery.tsx`.
